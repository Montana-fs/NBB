# NBB CRM — Installation og brug

## Start systemet

Dobbeltklik på filen **start.command** i CRM-mappen.

Browseren åbner automatisk på http://localhost:5002

Første gang oprettes der tre demo-kunder med aktiviteter og opkald, så du kan se systemet i brug med det samme.

## Stop systemet

Luk det sorte terminal-vindue der åbnede, eller tryk Ctrl+C i det.

## Hvis browseren ikke åbner selv

Åbn manuelt: http://localhost:5002

## Hvis der kommer en fejl første gang

macOS kan spørge om tilladelse til at køre filen. Højreklik på **start.command** → Åbn → Åbn alligevel.

## Dine data

Alle data gemmes i filen `data/crm.db` i CRM-mappen. Tag en kopi af den fil for at sikkerhedskopiere.

## Krav

- macOS
- Python 3 (hentes gratis fra https://www.python.org hvis ikke installeret)
